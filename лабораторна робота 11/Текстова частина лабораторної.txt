Лабораторна робота 11
1.Інформаційна система
Опис обраного проекту:
У даній лабораторній роботі необхідно проектувати інформаційну систему Вищого навчального закладу, у нашому випадку університету. 
Маємо певні категорії людей, що приймають участь у навчальному процесі. 
Студенти, організовані в групи, навчаються на факультеті, яким керує декан. 
У навчальному процесі беруть участь викладачі кафедр, вони поділяються на асистентів, викладачів, старших викладачів, доцентів та професорів. 
Асистенти і викладачі можуть навчатись в аспірантурі, ст. викладачі, доценти можуть очолювати наукові теми, професори-наукові напрямки.
Навчальний план містить в собі які навчальні дисципліни на якому з курсів викладаються для студентів,
 зазначається: кількості годин на кожен вид занять з дисципліни (види занять: лекції, семінари, лабораторні роботи, консультації, курсові роботи, ІР і т.д.) і форми контролю (залік, іспит).

Розробимо базу даних, де будуть зберігатись наступні відомості:
таблиця Студент( поля ID студента, код студента, ім'я, прізвище,
по батькові,група, курс, спеціальність,факультет, дата вступу, 
форма навчання(Денна/Заочна);
Таблиця Академічної успішності студентів( поля ID заліковки, семестр,
предмет,оцінка, дата екзамену, викладач);
Таблиця Викладачів( поля ID викладача, код викладача, ім'я, прізвище,
по батькові, факультет, учене звання);
Таблиця Спеціальності ( поля ID спеціальності, назва спеціальності);
Таблиця Факультет( поля ID факультету, назва факультету);
Таблиця Учене звання( поля ID звання, назва звання);
Таблиця Предмет( поля ID дисципліни, назва дисципліни);

Користувач Бази Даних має доступ до всієї інформації. Основними користувачами 
системи будуть: 
-Споживачі інформації: студенти, викладачі, декани, зав. Кафедрами;
-Колектив фахівців: адміністратор, системні програмісти, прикладні програмісти, аналітики;
-Веб-клієнти: студенти, які отримують частину інформації приватного характеру в особистому кабінеті, співробітники вузу-отримання певних довідок,інформації.
